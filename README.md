# 超能女人用超能 H5

这是根据 Figma 设计稿整理的 H5 静态页面项目。

Figma 设计稿：
https://www.figma.com/design/6AQ2X0V88oJ1omX32e1poF/%E8%B6%85%E8%83%BD%E5%A5%B3%E4%BA%BA%E7%94%A8%E8%B6%85%E8%83%BD?node-id=12-2

## 本地文件

- `index.html`：H5 页面入口
- `styles.css`：移动端画布和交互样式
- `script.js`：点击切换、图片预加载和语音播放逻辑
- `assets/prologue/`：早期序幕 PNG 素材
- `assets/figma-pdf/`：从 Figma 导出的 19 个完整 H5 画板 PDF

## 画板范围

Figma 中的 `H5部分` 画布包含封面、序幕、翻牌互动、第一幕和结尾画板。当前本地项目已包含 19 个完整画板文件，并保留点击切换和配音交互。
